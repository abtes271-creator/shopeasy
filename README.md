# 🛍️ ShopEasy — Modern E-Commerce Store

A full-featured e-commerce web app built with React, featuring product browsing, search, filtering, a persistent shopping cart, and a complete checkout flow.

**🔗 Live Demo:** https://shopeasy-topaz-eight.vercel.app/products

## Features

- 🛍️ Browse 20+ real products from a live API
- 🔍 Live search and category filtering
- 🛒 Shopping cart with quantity controls, powered by Context API
- 💾 Cart persists across page refreshes via localStorage
- ✅ Multi-step checkout with form validation
- 📱 Fully responsive — works on mobile, tablet, and desktop
- 🚀 Deployed on Vercel

## Tech Stack

- **React JS** — component-based UI
- **React Router** — client-side routing across 6 pages
- **Context API** — global cart state management
- **FakeStore API** — real product data
- **localStorage** — cart persistence

## Pages

| Route            | Description                                        |
| ---------------- | -------------------------------------------------- |
| `/`              | Home with hero section and featured products       |
| `/products`      | Full product grid with search and category filters |
| `/product/:id`   | Product detail page with quantity selector         |
| `/cart`          | Shopping cart with editable quantities             |
| `/checkout`      | Shipping form with validation                      |
| `/order-success` | Order confirmation page                            |
